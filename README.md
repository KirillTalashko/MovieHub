# 🎬 Movie Hub

Movie Hub is a modern Android application for discovering and browsing movies and TV series.

The app allows users to explore featured content, search for movies, and view detailed information.

---

## 🚀 Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- Clean Architecture
- Dagger 2
- Coroutines + Flow
- Retrofit
- Room

---

## 🏗 Architecture

The project uses a modular Clean Architecture approach:

- `domain` – business logic
- `data` – repository implementations
- `core` – shared infrastructure
- `feature-*` – UI screens

---

## 📱 Features

- Home screen with featured movies
- Search functionality
- Movie details screen
- Favorites management

---

## 🛠 How to Run

1. Clone the repository  
2. Open in Android Studio  
3. Run the app  

---

## 🎯 Purpose

This project was built to practice scalable Android architecture and modern development tools.
